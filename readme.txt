GameEngineTutorial
1. Install NPM
2. Install Phaser
	npm install phaser@v3.80.1
3. Install webserver
	npm install http-server
4. Start webserver
	npm start
5. Open either Edge or Google Chrome in incognito mode
6. Go to this address in browser
	http://localhost:8000/
7. Refresh the page to see live changes
8. When done, just close the webserver
